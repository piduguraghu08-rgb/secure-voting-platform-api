# Secure Voting Platform API

## Overview
A secure backend voting system built using Spring Boot, JWT authentication, and MySQL.

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Maven

## Features
- Secure user authentication
- Vote casting
- Duplicate vote prevention
- Vote result aggregation
- RESTful APIs

## Setup
1. Create database:
   CREATE DATABASE votingdb;
2. Update application.properties
3. Run:
   mvn clean install
   mvn spring-boot:run
